# Collaboration and Sharing

<Note>
Effective collaboration and secure sharing of information are fundamental in modern design and production environments. This section explores the application's features dedicated to collaboration, revision management, and data sharing.
</Note>

## Overview of Collaboration Features

<AccordionGroup>
  <Accordion title="Collaborative Architecture" icon="network-wired">
    The application offers a complete ecosystem for team collaboration:
    - **Real-time collaboration**: Multiple users can work simultaneously on the same project
    - **Cloud synchronization**: Access projects from any device and location
    - **Permission system**: Granular control over who can view or modify what
    - **Complete history**: Track all changes with rollback capability
    - **Smart notifications**: Targeted updates on relevant changes
  </Accordion>

  <Accordion title="Multidisciplinary Integration" icon="users-gear">
    Support for different team roles:
    - **CAD designers**: Creation and modification of geometry
    - **CAM programmers**: Toolpath generation
    - **Process engineers**: Production parameter definition
    - **Production staff**: Access to manufacturing information
    - **Managers**: Progress and timeline supervision
  </Accordion>
</AccordionGroup>
